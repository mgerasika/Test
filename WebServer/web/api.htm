Calibrate $arduino.calibrate(); This command move burn pen into top left corner.This commands turns x and y engines.Stops when buttons pressed.
Green Led On/Off (13pin) $arduino.writeToComPort(msg); Set 13 pin on arduino to HIGH or LOW state(turn on/off internal led)
Motor X left $arduino.onMotorX("left"); $arduino.offMotorX(); Start rotating X motor,and moving fire pen to left.
Motor X right $arduino.onMotorX("right"); $arduino.offMotorX(); Start rotating X motor,and moving fire pen to right.
Motor Y up $arduino.onMotorY("up"); $arduino.offMotorY(); Start rotating Y motor,and moving fire pen to up.
Motor Y down $arduino.onMotorY("down"); $arduino.offMotorY(); Start rotating Y motor,and moving fire pen to down.
Motor Z up $arduino.onMotorZ("up"); $arduino.offMotorZ(); Start rotating Z motor.
Motor Z down $arduino.onMotorZ("down"); $arduino.offMotorZ(); Start rotating Z motor.
Move to XY $arduino.moveXY(x,y); Moving fire pen to x,y coordinate.
Draw point $arduino.drawPoint(x,y); draw point intox x,y coordinate.
Draw line $arduino.drawLine(x1,y1,x2,y2); Draw line.
